Drag & drop or click to upload

🛡️ Zero-Knowledge Security

ZeroG Encryption runs entirely locally within your browser using the hardware-accelerated Web Crypto API. The decryption key is derived locally and is never sent over any network.

Under the Hood:
• Algorithm: AES-256-GCM (Galois/Counter Mode).
• Key Derivation: PBKDF2 with HMAC-SHA256 and 100,000 iterations.
• Cryptographic Salt: Generated uniquely for each file (16 bytes).
• Initialization Vector: Randomly generated for every session (12 bytes).
⚠️ WARNING: There is no password recovery. If you forget your password, your file cannot be decrypted.
🔓

Choose Encrypted File

Drag & drop or click to upload

📂 Restoring Encrypted Files

Upload a file ending in .enc, type the correct password, and your browser will recreate the original unencrypted file immediately for download.

Verification Steps:
1. Reads the salt & IV prepended to the binary data.
2. Regenerates the AES-256 decryption key from the password.
3. Decrypts and validates the authentication tag.
4. Prompts you to download the original file.

🔒 Privacy Policy

100% Client-Side Privacy Commitment

ZeroG Toolbox operates entirely in your web browser. All AI models (Qwen2.5, Whisper, MODNet), image manipulations, optical character recognition (OCR), audio transcriptions, and file cryptographic decryptions are executed locally on your device via WebGPU and WebAssembly. No data, files, text inputs, or tokens are ever uploaded, transmitted, or stored on external servers.

Local Analytics

This application does not track your personal identity or use tracking cookies. Your browser's localStorage is used solely to persist user interface state (such as light/dark mode preference or temporary local workspace states) for convenience.
